# Game Menu Demo
A ultra simple game menu for Unity3D to use during your next game jam.

# Requirements
 - Unity3d

# Features
 - Play: load the main game scene 
 - Credits: load a menu section to exhibit the team who developed the game
 - Exit: quit the application
